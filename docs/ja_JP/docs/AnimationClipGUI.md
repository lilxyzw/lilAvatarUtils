# アニメーション

アバターに含まれる全てのアニメーションの一覧です。アニメーションに含まれるキーフレームの種類も確認できます。

![AnimationClipGUI](/images/ja_JP/AnimationClipGUI.png "AnimationClipGUI")
## 使い方

エディタ上で発生せずゲーム上で発生するトラブルは多くの場合アニメーションが原因ですが、アニメーション数が多いと原因の特定に時間がかかります。このツールではアバターに含まれる全アニメーションを列挙し、さらにアニメーションの種類も表示することで原因となっているアニメーションを特定するのに役立ちます。

## プロパティ

|名前|説明|
|-|-|
|名前|アセット名です。クリックするとProjectウィンドウ上で該当アセットを選択できます。|
|Humanoid操作|アニメーションがHumanoidの操作を含むかどうかです。|
|ブレンドシェイプ操作|アニメーションがブレンドシェイプの操作を含むかどうかです。|
|オブジェクトのオンオフ|アニメーションがオブジェクトのオンオフを含むかどうかです。|
|コンポーネントのオンオフ|アニメーションがコンポーネントのオンオフを含むかどうかです。|
|Transform操作|アニメーションがTransformの操作を含むかどうかです。|
|マテリアル置き換え|アニメーションがマテリアルの置き換えを含むかどうかです。|
|マテリアルプロパティ操作|アニメーションがマテリアルプロパティ操作を含むかどうかです。|
|その他|アニメーションがそれ以外の操作を含むかどうかです。|
